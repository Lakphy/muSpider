## 参与适配榜单

您不需要会编程即可参与适配

如果想要参与适配，您需要具备以下能力：

- 会进行网络抓包
- 能够分析抓包内容

仅需要这两样能力，您便可以贡献适配榜单

目前，我可以为您提供以下信息以减轻贡献适配榜单的工作量：

- 目标网页 [https://y.qq.com/n/yqq/toplist/4.html](https://y.qq.com/n/yqq/toplist/4.html)
- 包含榜单音乐id的文件为 musicu.fcg文件
- 上述大约在网页打开后3000-5000ms内出现
